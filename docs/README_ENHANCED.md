## üöÄ Quick Start (Enhanced)

### 1. Clone and Setup
```bash
git clone https://github.com/lucylow/autorl-arm-edition.git
cd autorl-arm-edition

# Install dependencies
pip install -r requirements.txt
pip install -r requirements-arm.txt
```

### 2. Run Interactive Demos
```bash
# Start frontend with demos
cd autorl_project/autorl-frontend
npm install
npm start

# Open http://localhost:3000/demo
# Try interactive ARM demos!
```

### 3. Benchmark Your Device
```bash
# Run ARM benchmarks
python scripts/arm/benchmark_device.py

# See results:
# - Inference latency
# - ARM feature detection
# - Performance comparison
```

### 4. Integrate into Your App
```python
from autorl_project.src.arm import get_arm_inference_engine

engine = get_arm_inference_engine()
result = engine.run_vision_inference('image.jpg')

print(f"Latency: {result['latency_ms']}ms")
print(f"Device: {result['device']}")
```
